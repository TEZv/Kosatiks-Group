K Mentorship Hub

A selective mentorship beta built around clarity, curation, and lightweight intake. Not a full Discord community, not a talent pool, and not a giant academy from day one.

Launch notes

Current status

Parked betaApplication-firstNo Discord-first ops

This hub only makes sense if it stays light. One page, one intake, one resource layer, and a clear selection rule. Community scale should come later, if it is earned.

What this should be now

  • one explanation page
  • one application form
  • one curated resource map
  • one simple follow-up flow

What belongs later

  • cohorts after selection
  • private Discord or Circle space
  • optional accountability quests
  • networking layer if demand is real

What not to build now

  • full academy infrastructure
  • many psychological tests
  • news parsers and moderation bots
  • high-touch operations without capacity

Recommended route

Use a simple application-first beta: form -> manual review -> selected follow-up. Link to existing resources such as roadmap-style materials instead of rebuilding an entire academy.
